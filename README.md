You can **download the entire repository content** via the following link:

+ [Download all files in this repo ~ 57.2 MB](https://github.com/vpnry/paauk/archive/refs/heads/main.zip)

> For meditation Chart/Map Files, please download them [from this repo](https://github.com/vpnry/paauk-chart).
